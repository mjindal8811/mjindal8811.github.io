Projects

WISC-S25 Pipelined Processor

Verilog ModelSim Computer Architecture  • April 2025

  • Designed a 16-bit single-cycle processor for the WISC-S25 ISA, supporting compute, memory, and control operations.
  • Built a 5-stage pipelined CPU with hazard detection, data forwarding, and register bypassing for efficient execution.
  • Integrated 2 KB I-cache and D-cache (2-way set-associative, 16 B blocks) with custom controllers and write-through, write-allocate policies.
  • Implemented pipelined memory access, cache-miss FSMs, and arbitration logic for seamless main-memory interaction.

Whack-a-Mole (Custom PCB + Firmware)

Altium Designer ARM Cortex-M Embedded C  • April 2025

  • Designed a custom PCB in Altium, integrating an Infineon-based ARM Cortex-M MCU for a responsive game platform; emphasized meticulous schematic capture and layout optimization.
  • Developed firmware in ModusToolbox to handle gameplay, peripherals, and timing—leveraging vendor documentation for precise component configuration.

xv6 Multi-Page Memory Management

C xv6 Paging & Syscalls  • March 2025

  • Extended xv6 to support user-level allocation of both 4 KB and 4 MB pages with isolated pools and dynamic reuse logic.
  • Integrated Transparent Huge Page (THP) behavior with syscall toggles and size-aware allocation policies for malloc.
  • Enhanced virtual-to-physical mapping and memory subsystem for multi-size pages, adding alignment safeguards, PDE-level changes, and kernel-side safety checks.

Knight’s Tour (High-Speed Digital System)

SystemVerilog Synopsys Tools DE0 FPGA ModelSim/Questa  • Dec 2024

  • Achieved 333 MHz by tightening area and timing through targeted synthesis strategies; used coverage analysis to refine tests and boost reliability with post-synthesis simulations.
  • Engineered multiple SystemVerilog blocks and integrated SPI/UART peripherals; verified with rigorous testbenches and real-time debug on a DE0 FPGA board.

Press-Your-Luck (Dual-Board RTOS Game)

PSoC Creator ModusToolbox CYHAL APIs FreeRTOS  • Nov 2024

  • Built an interactive game on a custom ECE 353 board (PSoC6, Cortex-M), managing peripherals from board/company schematics and CYHAL docs; optimized responsiveness with interrupt-driven logic.
  • Integrated FreeRTOS for robust HW/SW synchronization; implemented SPI, I2C, and UART between dual boards to enable a multiplayer experience.
